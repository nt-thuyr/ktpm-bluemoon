# bluemoon

1. how to run the project
```bash
# Clone the repository
# Create a virtual environment (1 time setup)
python -m venv venv
# Activate the virtual environment
.\venv\bin\activate  # path to activate script may vary based on OS
# Install dependencies
pip install -r requirements.txt
# Run the project
python .\src\backend\run.py
```
Go to http://127.0.0.1:5000/ in your browser to see the application running.
